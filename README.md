# 🎖️ JARVIS — 나를 위한 개인 관리사

군 생활 중 **피부·몸·공부** 등 여러 고민을 한 곳에서 관리하고 코칭받는 개인 대시보드입니다.
설치·로그인·서버 없이 브라우저에서 바로 열립니다. 모든 기록은 내 브라우저(localStorage)에
저장돼서 껐다 켜도 남아요.

## 사용법

`index.html` 파일을 브라우저로 열기만 하면 됩니다.

```
open index.html      # macOS
xdg-open index.html  # Linux
```

휴대폰에서 쓰려면 파일을 옮겨 브라우저로 열거나, 나중에 GitHub Pages로 배포하면 됩니다.

## 기능

- 🏠 **홈 / 브리핑** — 오늘 상태 요약, **전역 D-day**(전역일 클릭해 설정), 나만의 목표/명언
- ✅ **오늘의 루틴** — 매일 자정 초기화되는 체크리스트, 자유롭게 추가/삭제
- 💪 **몸** — 팔굽혀펴기·윗몸·스쿼트 누적, 달리기(km)·몸무게 기록과 추이
- 📚 **공부** — 집중 타이머(오늘/누적 시간 저장), 할 일 목록
- 🌸 **피부** — 아침/저녁 스킨케어 체크, 피부 진단으로 맞춤 성분 가이드
- 📔 **일지** — 기분 + 하루 한 줄 기록

## 데이터

- 모든 데이터는 **이 브라우저 안에만** 저장됩니다 (localStorage, 키: `jarvis_v1`).
- 서버로 아무것도 전송하지 않습니다. 브라우저 데이터를 지우면 기록도 사라집니다.
