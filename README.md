# 🌸 스킨케어 어드바이저 (나만의 피부 관리사)

설치 없이 브라우저에서 바로 열리는 단일 페이지 웹앱입니다. 피부 타입과 고민을
설문으로 입력하면 맞춤 스킨케어 루틴(아침/저녁), 성분 가이드, 관리 팁을 보여줍니다.

## 사용법

`index.html` 파일을 브라우저로 열기만 하면 됩니다. 별도 서버나 빌드 과정이 필요 없습니다.

```
open index.html      # macOS
xdg-open index.html  # Linux
```

## 기능

- **피부 진단 설문**: 피부 타입 / 주요 고민(복수) / 예민도 / 연령대
- **맞춤 데일리 루틴**: 아침·저녁 단계별 추천 (클렌징 → 세럼 → 보습 → 자외선 차단)
- **성분 가이드**: 고민별 추천 성분 · 주의 성분
- **관리 팁**: 생활 습관 포함 실천 팁

모든 추천 로직은 `index.html` 안의 JavaScript 규칙 기반으로 동작하며, 외부 API나
네트워크 연결이 필요 없습니다.

## 확장 아이디어

- Claude API 연동으로 자유 서술형 상담 챗봇 추가
- 백엔드(REST API)로 진단 이력 저장
- 제품 데이터베이스 연동한 실제 제품 추천

> 피부과 전문의와 상담하세요.
